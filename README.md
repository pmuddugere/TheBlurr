### Product Fields
Each product includes:
- `id`: Unique identifier (e.g., PROD001)
- `name`: Product name
- `category`: Necklaces, Rings, Earrings, Bracelets
- `collection`: Collection name
- `price`: Price in INR
- `originalPrice`: Original price (if on sale)
- `currency`: Base currency symbol
- `rating`: 1-5 stars (supports half stars)
- `reviews`: Number of reviews
- `badges`: Array of badges (New Arrival, Bestseller, Limited Edition)
- `shortDescription`: Brief description for cards
- `description`: Full detailed description
- `images`: Array of image filenames (4 per product)
- `specifications`: Object with metal, stones, dimensions, weight, etc.
- `availability`: "In Stock" or "Limited Stock"
- `sku`: Stock Keeping Unit code
- `inStock`: Boolean
- `lowStock`: Boolean (triggers urgency messaging)
- `limitedEdition`: Boolean
